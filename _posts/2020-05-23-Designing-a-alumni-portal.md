## Introduction

The tool was required to be built for the IIITD alumnus. The tool was intended to replace the existing paid service, and to make an in-house solution for the same. The tool had the following pages.
- Home
- Events
- Careers
- Donations
- Contact
- Donate



![Alluvial Diagram](/assets/img/alumni-1.jpeg)

## Tech Stack

- Python 3
- Mongo DB
- PyMongo
- Flask
- HTML/CSS
- Pandas



The authentication was done using the Google mail authentication.
